About This Site

This is a demo of how to insert custom content sections on a website, hosted on GitHub Pages.

Custom Content Section

This section is designed to embed custom HTML and CSS from external sources.

How to Use the Custom Section

Method 1: Direct HTML Injection

Inject custom HTML directly into the content area. Best for static content such as text and images.


                <div class="custom-content-area">
                <!-- Your custom HTML here -->
                </div>

Method 2: Direct HTML Injection with iframe

Embed an iframe directly within the custom content area. Best for displaying external content.


                <div class="custom-content-area">
                <iframe src="https://example.com"></iframe>
                </div>

Method 3: JavaScript Loading

Use JavaScript to dynamically load content into the custom content area. Best for interactive or frequently changing content.


                document.getElementById('customContentArea').innerHTML = 
                '<div>Your custom HTML here</div>';
